# Smart Portfolio Generator
A web-based application designed to dynamically generate a professional personal portfolio from user-provided inputs. 
This tool enables users to quickly create a structured and visually appealing portfolio without requiring prior web 
development experience.

##  Live Application
https://smartportfoliogenerator.netlify.app

##  Overview
The Smart Portfolio Generator simplifies the process of building a personal portfolio by converting user inputs into a 
fully structured web interface. It demonstrates practical implementation of frontend development concepts including DOM
manipulation, event handling, and dynamic rendering.

## ✨ Key Features
- Dynamic portfolio generation using real-time user input  
- Structured sections for About, Skills, Education, Projects, and Contact  
- Interactive and responsive user interface  
- Profile image upload functionality  
- Clickable contact links (email and phone integration)  
- External profile linking (GitHub and LinkedIn)  
- Smooth navigation with section-based scrolling  
- Deployed and publicly accessible via Netlify  

##  Technology Stack
**Frontend:** HTML, CSS, JavaScript  
**Deployment:** Netlify  

##  Application Workflow
1. User provides personal and professional details via input form  
2. JavaScript processes and structures the data  
3. Portfolio content is dynamically rendered on the UI  
4. Generated portfolio is displayed instantly with navigation support
